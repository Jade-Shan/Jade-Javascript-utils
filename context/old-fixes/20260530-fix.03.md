### 问题 1：硬编码中文提示文案（中等）

**文件**: `web.ts:359`

`checkImageFileSize` 方法中使用了硬编码的 `alert()` 中文提示。作为通用工具库，应让调用者决定提示方式和文案。

**原代码**:
```typescript
static checkImageFileSize(fileInput: HTMLInputElement, imgMaxSize: number): boolean {
    if (fileInput.files && fileInput.files[0]) {
        console.log(fileInput.files[0].size);
        if (fileInput.files[0].size > imgMaxSize) {
            alert(`图片大于${Math.round(imgMaxSize / 1024)}K，请压缩后上传`);
            return false;
        }
        let fileList: FileList = fileInput.files;
        for (let i = 0; i < fileList.length; i++) {
            let file: (File | null) = fileList.item(i);
            if (file && file.size > imgMaxSize) {
                alert(`图片大于${Math.round(imgMaxSize / 1024)}K，请压缩后上传`);
                return false;
            }
        }
    } else {
        console.error("browser version too old");
        return false;
    }
    return true;
}
```

**建议修复**:
```typescript
static checkImageFileSize(fileInput: HTMLInputElement, imgMaxSize: number,
    onOversize?: (file: File, maxSize: number) => void): boolean {
    const handleOversize = onOversize ?? ((file, max) => {
        alert(`图片大于${Math.round(max / 1024)}K，请压缩后上传`);
    });
    if (fileInput.files && fileInput.files[0]) {
        let fileList: FileList = fileInput.files;
        for (let i = 0; i < fileList.length; i++) {
            let file: (File | null) = fileList.item(i);
            if (file && file.size > imgMaxSize) {
                handleOversize(file, imgMaxSize);
                return false;
            }
        }
    } else {
        console.error("browser version too old");
        return false;
    }
    return true;
}
```

### 问题 2：`checkImageFilePostfix` 正则遗漏 webp/svg 格式（低）

**文件**: `web.ts:344`

现代浏览器普遍支持 webp 和 svg 格式的图片，正则中缺少对这些格式的匹配。

**原代码**:
```typescript
static checkImageFilePostfix(postfix: string): boolean {
    return /\.jpg|\.gif|\.png|\.bmp/i.test(postfix);
}
```

**建议修复**:
```typescript
static checkImageFilePostfix(postfix: string): boolean {
    return /\.(jpg|jpeg|gif|png|bmp|webp|svg)$/i.test(postfix);
}
```

> 注意：原正则还有一个潜在问题——`/\.jpg|\.gif|\.png|\.bmp/` 等价于 `/(\.jpg)|(\.gif)|(\.png)|(\.bmp)/`，匹配任何包含这些子串的字符串（如 `file.jpg.exe` 也会通过）。修复版使用 `$` 锚定末尾，更严格。

### 问题 3：`NumUtil.div` 对特殊输入缺少保护（低）

**文件**: `basic.ts:151`

当除数 `n2 = 0` 时，方法内部 `r1 / r2` 会产生 `Infinity` 或 `NaN`，且 `toString().replace(".", "")` 对科学计数法（如 `1e-7`）表现不正确。

**原代码**:
```typescript
static div(n1: number, n2: number): number {
    let t1 = this.getDecimalPlaces(n1);
    let t2 = this.getDecimalPlaces(n2);
    let m = t2 - t1;
    let r1 = Number(n1.toString().replace(".", ""));
    let r2 = Number(n2.toString().replace(".", ""));
    return (r1 / r2) * Math.pow(10, m);
}
```

**建议修复**:
```typescript
static div(n1: number, n2: number): number {
    if (n2 === 0) return n1 >= 0 ? Infinity : -Infinity;
    let r1 = this.getDecimalPlaces(n1);
    let r2 = this.getDecimalPlaces(n2);
    let m = Math.pow(10, Math.max(r1, r2));
    return (Math.round(n1 * m) / Math.round(n2 * m));
}
```

### 问题 4：`webHtmlPage.ts` 中 `renderSubTitle` 使用 `innerHTML` 设置纯文本（低）

**文件**: `webHtmlPage.ts:230`

`subTitle` 是配置文本，用 `innerHTML` 赋值存在 XSS 风险（如果 subTitle 来源不可信）。

**原代码**:
```typescript
renderSubTitle(elemSlt: string = "#subTitle"): void {
    let elem = document.querySelector(elemSlt);
    if (elem) {
        elem.innerHTML = this.cfg.subTitle;
    }
};
```

**建议修复**:
```typescript
renderSubTitle(elemSlt: string = "#subTitle"): void {
    let elem = document.querySelector(elemSlt);
    if (elem) {
        elem.textContent = this.cfg.subTitle;
    }
};
```

### 问题 5：`blog.ts:117` 中 `renderArticle` 将 Markdown 转换后的 HTML 直接插入（低）

```typescript
let st = ShowdownUtils.makeHtml(atc.text);
let html = `
    <div class="item">
        ...
        <div class="body">${st}</div>
    </div>`;
```

**建议修复**:
```typescript
// 在 ShowdownUtils 中或渲染前，对 makeHtml 的结果做 sanitize
// 例如引入 DOMPurify：
// import DOMPurify from 'dompurify';
// let st = DOMPurify.sanitize(ShowdownUtils.makeHtml(atc.text));
```

> 注意：由于项目不使用打包器且依赖全局脚本加载，可考虑通过 `<script>` 引入 DOMPurify CDN，然后 declare 全局变量使用。
